# Introdoction

Bioinformatics support for vim.

# Install

The recommended way is to use [pathogen][], [vundle][] or perhaps even [unbundle][].

Otherwise you can just copy the files to their respective subdirectory into your .vim directory.

[pathogen]: https://github.com/tpope/vim-pathogen
[vundle]: https://github.com/gmarik/vundle
[unbundle]: https://github.com/sunaku/vim-unbundle
